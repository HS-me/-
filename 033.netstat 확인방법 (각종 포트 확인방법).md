netstat -anlp 명령어도 가능하다고 함.



TCP만 확인
**netstat -anp -tcp | grep LISTEN


UDP만 확인
**netstat -anp -udp | grep LISTEN


특정 포트만 확인
**netstat -ano | find "443"**

ESXi 하이퍼바이저에서는 netstat이 안되고 해당명령어 사용
esxcli network ip connection list | grep [tcp,udp,80 등등]
